# 📋 Decisiones de Arquitectura (ADRs)

Este directorio contiene los Architecture Decision Records (ADRs) del proyecto.

## Formato

Cada ADR sigue el formato:

```markdown
# ADR-XXX: Título de la Decisión

## Estado
[Propuesta | Aceptada | Deprecada | Reemplazada]

## Contexto
¿Cuál es el problema o la necesidad?

## Decisión
¿Qué decidimos hacer?

## Consecuencias
¿Cuáles son los efectos positivos y negativos?

## Alternativas Evaluadas
¿Qué otras opciones consideramos?
```
