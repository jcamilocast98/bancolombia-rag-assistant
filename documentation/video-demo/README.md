# 🎬 Video Demo

Esta carpeta está reservada para el video demostración del proyecto.

## Requisitos del Video

- **Duración**: 3-5 minutos máximo
- **Formato**: MP4 o enlace a YouTube/Loom

## Contenido Sugerido

1. **Introducción** (30s): Breve explicación del problema y la solución
2. **Arquitectura** (1min): Recorrido por los diagramas C4
3. **Demo en Vivo** (2min):
   - Mostrar el chat funcionando
   - Hacer preguntas sobre productos Bancolombia
   - Mostrar citación de fuentes
   - Mostrar historial de conversación
4. **Decisiones Técnicas** (1min): Justificación de tecnologías elegidas
5. **Cierre** (30s): Limitaciones y mejoras futuras
